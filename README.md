# 💵 API de Cotação do Dólar (Flask)

Este é um projeto simples de uma API desenvolvida em **Python com Flask** que consome uma API externa para retornar a cotação atual do dólar em relação ao real (USD/BRL).

---

## 🚀 O que o projeto faz

- Cria uma API local com Flask
- Busca dados de uma API externa de câmbio
- Retorna a cotação atual do dólar
- Expõe endpoints simples para consulta
